# Contributing to Pinnacle Data Platform

We welcome contributions to help build the future of data-centric AI tools!

## Getting Started

1.  Fork the repository.
2.  Clone your fork: `git clone ...`
3.  Run the stack: `docker-compose up`

## Development Guidelines

*   **Backend**:
    *   Follow PEP 8.
    *   Use Pydantic schemas for all API I/O.
    *   Ensure database models have corresponding migration scripts (Alembic).
*   **Frontend**:
    *   Use Functional Components with Hooks.
    *   Stick to Tailwind CSS for styling.
    *   Ensure strict typing with TypeScript.

## Feature Requests

Please check the `DESIGN.md` to align with the 2026 vision before proposing major architectural changes.
